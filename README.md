# FareCalc Travel Optimizer

## About the Project

This is a Python-based fare calculator that simulates how ride-sharing apps estimate prices.
The fare is calculated based on distance, vehicle type, and time of travel.

---

## Features

* Multiple vehicle options (Economy, Premium, SUV, Bike, Auto, Electric)
* Peak hour surge pricing (5 PM – 9 PM)
* Night charges for late hours
* Case-insensitive input (user can enter vehicle names in any format)
* Clean and structured price receipt

---

## Concepts Used

* Dictionary (for storing vehicle rates)
* Functions (for fare calculation logic)
* Conditional statements (for surge and night pricing)
* Exception handling (to manage invalid input)
